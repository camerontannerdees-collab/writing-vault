# Source Material

Raw inputs. Things I'd want to draw from but didn't write myself.

## What goes here

- Meeting notes and transcripts
- Articles and external writing
- Decks and documents converted to markdown
- Research notes, data dumps, conversation captures
- Anything that might be cited in a draft

## What doesn't go here

- My own finished writing (that's `docs/`)
- Working drafts (that's `docs/`)
- The writing guide or context (that's `context/`)

## Naming

`YYYY-MM-DD-short-description.md` — date is when the source was created, not when it was added.

## Citations

When the agent draws on a source file in a draft, it cites the file path inline. Every claim from a source should be traceable.
