## Capture

Drop raw notes, links, transcripts into `source-material/`. The agent will read them on demand and cite them when drafting.

## Workflow

1. Capture into `source-material/`
2. Open Claude Code in this folder (`claude` in the terminal)
3. Tell it what to draft or refine, pointing at the relevant sources
4. Review, correct, save corrections as feedback files when they should stick
