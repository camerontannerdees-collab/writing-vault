# Writing Index

All pieces, organized by topic. Each topic is a sub-folder. Drafts and published pieces live together within a topic — status is tracked in frontmatter or filename, not by folder.

## Naming convention

`YYYY-MM-DD-slug.md` for dated pieces (essays, posts, updates).
`slug.md` for evergreen pieces or works-in-progress that don't have a publish date yet.

## Status in frontmatter

```
---
status: draft | review | published
audience: public | investors | team | customers
created: YYYY-MM-DD
---
```

## Topics

Topics get created as the writing happens. The agent should suggest a new topic folder when a piece doesn't fit any existing one, not jam it into the closest match.
